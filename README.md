# Nemis Tili O'rganish Platformasi

## Maqsad
Bu loyiha Nemis tilini turli xil ko'rinishdagi materiallar orqali chuqur va samarali o'rganish uchun yaratilgan.

## Asosiy Xususiyatlar
- 📚 **Matn Darsliklar**: Kemahl структураланган dars materiallari
- 🎥 **Video Kurs**: Nemis tilida natiqlarni anglash kotirilgan videolar
- 🎵 **Audio Mashqlar**: Tilning to'g'ri talaffuzini o'rganish
- 📄 **PDF Resurslar**: Murakkab grammatika va lug'at mavzulariga oid kitoblar
- 🖼️ **Visual Darslar**: Rasmlar va sxemalar orqali lug'at o'rganish
- 📊 **Testlar va Quiz**: Bilik darajangizni tekshirish
- 💾 **Mahalliy Ma'lumotlar Tahlili**: Barcha materiallar to'plamini saqlash va o'rganish

## Texnologiyalar
- **Backend**: Python (Flask/Django)
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite/MySQL
- **Multimedia**: Audio/Video Processing
